Servidor FTP:

Um servidor FTP (File Transfer Protocol) é um tipo de servidor que permite a transferência de arquivos entre computadores pela internet ou uma rede local, utilizando o protocolo FTP. O FTP é um protocolo de rede padrão usado para transferir arquivos de um computador para outro, seja para upload (envio de arquivos) ou download (recebimento de arquivos).

JavaScript:

JavaScript é uma linguagem de programação de alto nível, amplamente utilizada no desenvolvimento web. Ela foi originalmente criada para ser executada em navegadores de internet, permitindo que páginas web se tornassem interativas e dinâmicas. Hoje, o JavaScript não se limita apenas a navegadores; ele também pode ser utilizado no desenvolvimento de aplicativos de servidor, aplicativos móveis, e muito mais.

Docker:

Docker é uma plataforma de software que permite automatizar a construção, o envio e a execução de aplicações dentro de contêineres. Os contêineres são ambientes isolados e leves, que contêm tudo o que uma aplicação precisa para ser executada — como o código, bibliotecas, dependências, e configurações do sistema. Isso garante que a aplicação funcione de forma consistente em diferentes ambientes, independentemente de onde ela seja executada.
